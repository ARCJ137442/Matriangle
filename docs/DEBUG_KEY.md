# Keys for Debugging

* `M`: Turn to **Menu**
* `G`: Turn to **Server**
* `R`: Change **Team(Color)**
* `T`: Transform **Pos/Map**
* `C`: Clear **Projectile/Effect**
* `V`: Change **Tool**
* `X`: Change **Sheet/I18ns**
* `~`: Manipulate **Server Speed**
* `N`: Append **Player**s
* `Enter`: Enter one **Server Step**
* `Shift+E`: Force get **Server Result**
